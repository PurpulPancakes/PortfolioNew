Python Games

Dragon Realm

Dragon Realm is a Python application where I learned how to use a randomizer and have messages appear after a little pause. In this project I coded it so that you can choose 1 of 2 options, and the correct choice is always randomized each time. After you have choosen a path to go down, you are introduced to a simple cutscene I created with the use of pausing the messages, which gives the viewer enough time to read before the next line shows up, as well as adding some suspence as to what will happen next!

Guess the Number

Guess the Number is a simple program I made in Python in which a random number is generated from 1-100, and you have a few tries to guess the number correctly, however after each guess you will be informed on if the random number is higher or lower than your choice, which can be used to create a fun short game!

Hangman

Hangman is the original hangman game that you know today, but recreated in a python script! It displays the drawing of your answers remaining, a randomized word to guess, already guessed letters, updating the blanks for correct answers, and a replay system to play another game! What I find the best about this is the fact that it's very easy to add extra words to the game, as they are all in an array, and you can simply add or remove words to your hearts content!

TIC TAC TOE

This is my first Python game that uses an algorithm! This is a recreation of the tic tace toe game, but you face a computer algorithm who tries to get three in-a-row before you do! The gameplay works the best with the numpad on most keyboards, as it has a three by three grid very similar to tic tac toe, and it's very easy to see where you will place your piece without counting!

TIC TAC TOE Multiplayer

Here is where I added a twist to my Tic Tac Toe Python game! This time it allows for 2 players (locally, litterally) to play together! Player 1 can choose X or O and can go first, in which instead of the computer taking it's turn, it allows for input again, where a second player can choose where to go! This was very exciting to get done, and it's fun to play with others!

Virtual Basic Games

The Quest for the Grilled Cheese Sandwich

The Quest for the Grilled Cheese Sandwich is a very simple, yet effective game made in Virtual Basic! You follow the journey of a warrior or a wizard, in which each screen you must make a choice, with only 1 path leading you to the grilled cheese sandwich at the end of both the wizard and the warrior's path respectively. Can you find the grilled cheese sandwich and satisfy your hunger?

The Quest for the Chocolate Cake

The Quest for the Chocolate Cake is the sequel to The Quest for the Grilled Cheese Sandwich! This game was created with Visual Basic instead, which allowed for the game to be much smoother than the original, and made routing quite easy! This time there are much options to select, but as you play as both the Wizard and the Warrior together, only 1 path is correct! Can you find the chocolate cake and have a wonderful dessert of friendship!?
